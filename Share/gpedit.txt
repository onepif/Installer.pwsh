Disable MSBallPoint:

gpedit.msc -> Политика "Локальный компьютер" -> Административные шаблоны -> Система -> Установка устройства -> Ограничения на установку -> Запретить установку устройств с указанными кодами устройств -> "Включить" -> Показать ->
вбиваем код: *PNP0f09

gpedit.msc -> Политика "Локальный компьютер" -> Административные шаблоны -> Система -> Установка устройства -> Отключить всплывающее напоминание "Найдено новое оборудование" во время установки драйвера -> Включить

Disable USB massstorage devices:
gpedit.msc -> Политика "Локальный компьютер" -> Административные шаблоны -> Система -> Доступ к съемным запоминающим устройствам -> Съемные запоминающие устройства всех классов: Запретить любой доступ -> Включить

Запрещаем пользователям менять время и дату:
Computer\Windows Settings\Security Settings\Local Policies\User Rights Assignment
свойство "Change the system time" - в нём перечисленны пользователи и группы которым можно менять время и дату

Настройка архивации Windows:
1. Панель управления - Архивация и восстановление;
2. Настройка архивации (выбрать место хранения образа);
3. Указать периодичность архивирования;
4. Ок.
